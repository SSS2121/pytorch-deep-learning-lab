# PyTorch Deep Learning Lab

Este repositorio contiene experimentos y arquitecturas de Redes Neuronales construidas con PyTorch, enfocadas en entender el comportamiento de los tensores y el proceso de optimización.

## Proyectos Destacados
- **Arquitecturas base:** Implementación de Redes Neuronales Densas (MLP).
- **Visión Artificial:** Modelos de Redes Neuronales Convolucionales (CNN) para clasificación.
- **Pipeline de Entrenamiento:** Implementación manual de bucles de entrenamiento (`forward`, `backward`, `optimizer.step()`).

## Tecnologías
- Python 3.x, PyTorch, NumPy, Matplotlib (para visualización de métricas).

## Lo que demuestra este repo
- Comprensión profunda del cálculo de gradientes.
- Habilidad para manipular tensores de alta dimensión para procesamiento de imágenes.
